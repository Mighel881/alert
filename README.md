# MyAlert
Loading Alert


Show Alert Dark
```Swift
MyAlert.view(Show: true)
//or
MyAlert.view(Show: true,status: .dark)
```

Show Alert Light
```Swift
MyAlert.view(Show: true,status: .light)
```
Hide Alert
```Swift
MyAlert.view(Show: false)
```
